# Automation

This folder contains Bash scripts used to automate operational tasks on atlas.

Phase 3 focuses on building small, useful scripts that help with Linux server administration.

The goal is not to create complex scripts immediately. The goal is to build practical automation that can be tested, documented, improved, and reused.

## Planned Scripts

| Script | Purpose | Status |
|--------|---------|--------|
| `system-health.sh` | Generate a basic system health report | Complete |
| `disk-check.sh` | Check disk usage and report status | Complete |
| `memory-check.sh` | Check memory usage and availability | Planned |
| `system-update.sh` | Assist with safe system updates | Planned |
| `backup-home.sh` | Create a simple timestamped backup | Planned |

## Automation Goals

The scripts in this folder should be:

- Simple
- Readable
- Tested on atlas
- Documented
- Safe to run
- Useful for real server administration
