# Phase 3 Plan: Bash Automation

## Goal

The goal of Phase 3 is to create practical Bash scripts for basic Linux server administration on atlas.

This phase is focused on automation, repeatability, and operational thinking.

## Scope

Phase 3 will include small scripts for:

- System health reporting
- Disk usage checks
- Memory usage checks
- System update assistance
- Simple backups

## Out of Scope

The following items are not part of this phase:

- Docker containers
- Cloud infrastructure
- Terraform
- GitHub Actions
- Ansible
- Kubernetes
- Advanced monitoring platforms

Those topics are planned for later phases.
