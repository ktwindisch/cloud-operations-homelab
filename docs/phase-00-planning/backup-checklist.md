# Backup Checklist

## Purpose

This checklist confirms that the Lenovo ThinkPad T530 can be safely wiped before installing Ubuntu Server.

Installing Ubuntu Server will replace the current operating system and erase existing data on the internal drive.

## Wipe Decision

The ThinkPad can be fully wiped.

There are no personal files, documents, downloads, browser bookmarks, photos, or recovery files that need to be preserved before installation.

## Pre-Wipe Checklist

| Item | Status |
|------|--------|
| Personal files checked | Complete |
| Important documents checked | Complete |
| Photos checked | Complete |
| Browser data checked | Not needed |
| Software/license keys checked | Not needed |
| Windows recovery data needed | No |
| Before photo captured | Complete |
| Disk wipe approved | Yes |

## Risk

The main risk is permanent data loss from the existing operating system.

This risk is accepted because the laptop is being intentionally repurposed for the Cloud Operations Homelab.

## Decision

Proceed with wiping the internal drive during Ubuntu Server installation.

## Notes

This laptop has not been used regularly for approximately a year and a half.

The machine is being intentionally repurposed as a dedicated Linux server.
