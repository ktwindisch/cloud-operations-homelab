# Hardware Baseline

## Device

Lenovo ThinkPad T530

## Purpose

This laptop will be repurposed into a headless Ubuntu Server for the Cloud Operations Homelab project.

The server will be managed remotely through SSH from a separate workstation.

## Hardware Specifications

| Component | Details |
|----------|---------|
| Model | Lenovo ThinkPad T530 |
| CPU | Intel Core i7-3520M |
| Clock Speed | 2.90 GHz |
| Cores | 2 cores |
| Logical Processors | 4 |
| Memory | 16 GB RAM |
| Storage | 238 GB SSD |
| Network | Wi-Fi |
| Planned Operating System | Ubuntu Server |
| Planned Access Method | Remote SSH |

## Current State

The laptop has not been used regularly for approximately a year and a half.

Before being repurposed, it was used as a study machine during my transition into technology.

## Planned Role

The ThinkPad T530 will become the primary local server for this project.

It will eventually support:

- SSH administration
- Linux user and permission management
- Bash automation scripts
- Docker containers
- Monitoring services
- System logs
- Scheduled jobs
- Infrastructure automation
- Cloud integration experiments

## Notes

This machine has enough memory and storage to support the early and mid-stage phases of the homelab.

The CPU is older, but it is sufficient for Linux administration, lightweight Docker workloads, monitoring tools, and automation practice.
